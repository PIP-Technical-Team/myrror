# Extract Different Values - Internal

Extract Different Values - Internal

## Usage

``` r
extract_diff_int(myrror_object = NULL, tolerance = 0.0000001)
```

## Arguments

- myrror_object:

  myrror object

- tolerance:

  numeric, default to 1e-7

## Value

list with two elements:

1.  diff_list

2.  diff_table
