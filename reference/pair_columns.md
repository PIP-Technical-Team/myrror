# Pairs columns and prepares them for comparison.

Pairs columns and prepares them for comparison.

## Usage

``` r
pair_columns(merged_data_report, suffix_x = ".x", suffix_y = ".y")
```

## Arguments

- merged_data_report:

  joined prepared_dfx and prepared_dfy.

- suffix_x:

  suffix for dfx (default .x)

- suffix_y:

  suffix for dfy (default .y)

## Value

paired_columns
