# Get keys or default. A simple function wrapper which returns 'rn' (row names) if the data.table has no keys.

Get keys or default. A simple function wrapper which returns 'rn' (row
names) if the data.table has no keys.

## Usage

``` r
get_keys_or_default(keys, default = "rn")
```

## Arguments

- keys:

  character vector

- default:

  character

## Value

character
